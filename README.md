## ✨ Overview

**CodeCanvas** is a sleek, browser-based live code editor designed for developers, learners, and creators who want a fast and attractive environment to write and preview **HTML, CSS, and JavaScript** in real time.

Inspired by modern developer tools, CodeCanvas combines a clean interface with powerful features like **syntax highlighting**, **real-time preview**, **console output**, **file explorer sidebar**, **VS Code-style tabs**, **formatter tools**, **local auto-save**, and **smooth animated day/night switching**.

It is built using only **HTML, CSS, and JavaScript**, making it lightweight, fast, and easy to understand.

---

## 🚀 Features

- Live HTML, CSS, and JavaScript preview
- Syntax highlighting editor
- VS Code-inspired tabs and file explorer
- Built-in console panel
- Real ZIP export
- LocalStorage auto-save
- Day and night theme toggle
- Smooth floating UI animations
- Responsive split-screen layout
- Code formatting tools
- Copy current file and copy all code actions

---

## 🎯 Why CodeCanvas?

CodeCanvas is more than a simple code editor. It is a polished frontend developer tool that focuses on both **functionality** and **visual experience**.  
The goal is to make coding feel modern, smooth, and inspiring directly in the browser.
